QUICK START
================================================================================

# Local MongoDB Setup (Fastest)
1. Start MongoDB: brew services start mongodb-community
2. Clone repo: git clone <url>
3. Install: npm install
4. Init: npm run db:init && npm run db:seed
5. Run: npm run dev
6. Login: admin@womenskillhub.com / admin123

# MongoDB Atlas Setup
1. Create account at mongodb.com/cloud/atlas
2. Create free M0 cluster
3. Set MONGODB_URI in .env.local
4. Run: npm run db:init && npm run db:seed
5. Run: npm run dev
